#========================================================================================================================================================================================================================#
#======================================================================================================PREFACE===========================================================================================================#

The following report contains an analysis of weather data to deduce suitability to establish a temperature-controlled storage unit to store temperature-sensitive products between the cities of Braganca and Viseu in Portugal.


#========================================================================================================================================================================================================================#
#=====================================================================================================CONTENT============================================================================================================#

The file contains an overall report of the analysis, the weather data to the cities, the analysis code and the figures of the plots made during analysis.


#========================================================================================================================================================================================================================#
#=====================================================================================================USAGE==============================================================================================================#

The analysis is to find the most suitable location to set up a new Temperature-Controlled logistics hub in Portugal, which shall be used to store various goods such as Luxury Chocolates, Specialty Food, Pharmaceuticals, etc.

#========================================================================================================================================================================================================================#
#==================================================================================================REPLICATION===========================================================================================================#

1. Extract the contents of the .zip file into a new folder

2. Open the new folder, right-click anywhere and then click "Open in Terminal"

3. In the terminal,
	Run the command "python -m venv venv" if on Windows
	Run the command "python3 -m venv venv" if on MacOS

4. You will have a new folder within the analysis folder called 
"venv". Open it, then open scripts, and locate "Activate.ps1". Right Click on it and select "Run with PowerShell"

5. Close the venv folder, we have now successfully set up our virtual environment to run the analysis.

6.Open the code folder, then open the analysis file.
You may need to set up a kernel to run the file within your code editor. Just select on 'New Kernel' and then select the Venv kernel which pops up.

7. Run all code blocks to replicate the analysis.
